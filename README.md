# Jovia Platform

A full-stack influencer marketing and e-commerce platform connecting brands, influencers, and users. Built with React (Vite) and Django.

---

## 🔹 Key Features
- Multi-role system (Admin, Brand, Influencer, User)
- Product & event management
- Influencer collaboration system
- Chat and booking system
- Complaint management

---

## 🔹 Tech Stack
Frontend: React.js, Vite  
Backend: Django  
Database: SQLite  

---

## 🔹 Project Structure
Client/ → React frontend  
Server/ → Django backend  

---

## 🔹 Setup

Backend:
cd Server  
python -m venv venv  
venv\Scripts\activate  
pip install django django-cors-headers  
python manage.py runserver  

Frontend:
cd Client  
npm install  
npm run dev
